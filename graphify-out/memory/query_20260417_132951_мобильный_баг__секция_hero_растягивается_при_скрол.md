# Q: Мобильный баг: секция Hero растягивается при скролле из-за скрытия панели браузера

## Answer

Причина: window.innerHeight меняется когда скрывается нижняя панель браузера (back/forward). Решение: захватить window.innerHeight - 64 один раз при маунте через useState с lazy init, применить как height на мобиле. На десктопе md:min-h-[calc(100svh-4rem)]. Также: убран pt-16 из Hero, добавлен pt-16 в main App.tsx — нав теперь логически отделён от секции. Параллакс цветочков включён и на мобиле [0,-50] вместо [0,-90].
