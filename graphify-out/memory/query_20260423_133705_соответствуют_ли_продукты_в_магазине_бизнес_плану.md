# Q: Соответствуют ли продукты в магазине бизнес-плану?

## Answer

Ассортимент полностью соответствует бизнес-плану: Roses (Bestseller, 3.50 zł), Tulips (Seasonal, 2.90 zł), Peonies (Premium, 6.90 zł — по бизнес-плану тоже seasonal), Wildflowers (2.20 zł), Eustoma (4.50 zł), Chrysanthemum (3.20 zł). Единственное несоответствие: Peonies по бизнес-плану seasonal, но в магазине тег Premium. Диапазон цен 2.20–6.90 zł/стебель совпадает. Формат per-stem без букетов соответствует концепции.
